# React-Redux Assignment

## Getting Started

```bash
# clone repo - Use Git or checkout with SVN using the web URL
https://github.com/dashmeet17/react.git

# install dependencies
yarn

# start app and server by running below command
yarn run dev
```

1. node
2. npm
